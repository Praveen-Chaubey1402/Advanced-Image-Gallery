# 📸 Image_galary

A modern and responsive image gallery web application built using **HTML, CSS, and JavaScript**. The project includes image upload, image management, password-protected secret gallery, dark mode, search, filters, lightbox viewer, image sharing, downloading, and local storage support.

---

## 🚀 Features

### 📤 Image Upload
- Upload images from your device
- Instant image preview before upload
- Store uploaded images using Local Storage

### 🖼️ Gallery Management
- Add new images
- Edit image title and category
- Delete images
- Hide and unhide images

### 🔍 Search System
- Search images instantly by title or category

### 🎨 Category Filters
- View images by category
- Categories include:
  - Nature
  - City
  - Animals
  - Technology

### 🔒 Secret Gallery
- Password protected hidden gallery
- Hidden images are only visible after entering the correct password

### 🌙 Dark Mode
- Toggle between light and dark themes
- Theme preference saved automatically

### 🖼️ Lightbox Viewer
- Full-size image preview
- Previous and Next navigation
- Keyboard navigation support

### 📤 Share Images
- Share images using Web Share API
- Copy link fallback support

### ⬇️ Download Images
- Download images directly from the gallery

### ⛶ Fullscreen Mode
- View images in fullscreen mode

### 📊 Statistics Dashboard
- Total images count
- Hidden images count

### 💾 Local Storage
- Gallery data remains available after page refresh
